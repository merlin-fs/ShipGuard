
"Зачистка" локаций.

Карта состоит из нескольких комнат.
Некоторые комнаты могут открываться только по выполнению условий (например, убить N количчество мобов)
За убийство мобов дается exp, которое повышает уровень игрока.
С убитых мобов выпадает лут.
В каждой локации должен быть босс, с него должен выпадать луший лут.
Локация считается зачищенной полсе убийства босса.
